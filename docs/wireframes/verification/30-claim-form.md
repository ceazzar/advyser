A comprehensive multi-step form for advisors to submit their business claim with all required verification details. This form collects role information, validates business credentials, confirms proof of control, and gathers supporting evidence for AU regulatory compliance.

## API Endpoints

### Validate ABN
```
GET /api/v1/abn/validate/:abn
Response: {
  valid: boolean,
  abn: string,
  entityName: string,
  entityType: string,
  status: "active" | "cancelled",
  stateCode: string,
  postcode: string
}
```

### Lookup Adviser
```
GET /api/v1/credentials/lookup
Query: type, number
Response: {
  found: boolean,
  name: string,
  status: "current" | "suspended" | "cancelled",
  licensee: string,
  authorisations: string[]
}
```

### Send OTP
```
POST /api/v1/verification/send-otp
Body: { method: "phone" | "email", value: string }
Response: { sent: boolean, expiresAt: timestamp }
```

### Verify OTP
```
POST /api/v1/verification/verify-otp
Body: { code: string, claimId: string }
Response: { verified: boolean, error?: string }
```

### Submit Claim
```
POST /api/v1/claims
Body: {
  businessId: string,
  role: "owner" | "staff",
  ownerEmail?: string,
  businessInfo: { ... },
  credentials: { type: string, ... },
  verificationMethod: string,
  verificationCode: string,
  documents: [{ id, name, size }]
}
Response: {
  claimId: string,
  reference: string,
  status: "pending",
  submittedAt: timestamp
}
```

## AU Compliance Notes

### ABN Validation
- Source: Australian Business Register (ABR) API
- Validation: Checksum algorithm + live lookup
- Display format: XX XXX XXX XXX

### Financial Adviser Verification
- Source: ASIC Financial Advisers Register (FAR)
- Cross-reference: MoneySmart API
- Required fields: FAR number, AFSL number
- Status check: Must be "current" status

### Mortgage Broker Verification
- Source: ASIC Credit Register
- Required fields: Credit Rep number, ACL number
- Status check: Must be active registration

### Property/Buyer's Agent Verification
| State | Register | Licence Format |
|-------|----------|----------------|
| NSW | NSW Fair Trading | Numeric |
| VIC | Consumer Affairs Victoria | Alphanumeric |
| QLD | Office of Fair Trading | Numeric |
| WA | Department of Mines | Alphanumeric |
| SA | CBS | Numeric |
| TAS | CBOS Tasmania | Numeric |
| ACT | Access Canberra | Numeric |
| NT | NT Licensing | Alphanumeric |

### Data Retention
- Documents stored encrypted (AES-256)
- Deleted after claim approval (max 30 days)
- Audit log retained for 7 years (regulatory requirement)
