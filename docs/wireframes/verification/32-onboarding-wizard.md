## API Endpoints

### Get Onboarding State
```
GET /api/v1/onboarding/state
Response: {
  currentStep: 1 | 2 | 3 | 4,
  completed: boolean,
  profile: { ... },
  services: [ ... ],
  availability: { ... },
  plan: { ... }
}
```

### Update Profile (Step 1)
```
PUT /api/v1/onboarding/profile
Body: {
  displayName: string,
  position: string,
  bio: string,
  whoIHelp: string,
  languages: string[],
  photoUrl?: string
}
```

### Upload Photo
```
POST /api/v1/onboarding/photo
Body: FormData (file)
Response: {
  url: string,
  thumbnailUrl: string
}
```

### Update Services (Step 2)
```
PUT /api/v1/onboarding/services
Body: {
  services: [
    {
      name: string,
      description?: string,
      duration: string,
      price?: number,
      priceType: "fixed" | "from" | "poa"
    }
  ]
}
```

### Update Availability (Step 3)
```
PUT /api/v1/onboarding/availability
Body: {
  acceptingClients: boolean,
  serviceType: "local" | "national",
  serviceAreas?: string[], // postcodes
  offersOnline?: boolean
}
```

### Connect Calendar
```
POST /api/v1/onboarding/calendar/connect
Body: { provider: "google" | "outlook" }
Response: { authUrl: string }
```

### Select Plan (Step 4)
```
POST /api/v1/onboarding/plan
Body: { plan: "free" | "pro" | "team" }
Response: {
  success: boolean,
  checkoutUrl?: string, // for paid plans
  subscriptionId?: string
}
```

### Complete Onboarding
```
POST /api/v1/onboarding/complete
Response: {
  success: boolean,
  profileUrl: string,
  dashboardUrl: string
}
```
