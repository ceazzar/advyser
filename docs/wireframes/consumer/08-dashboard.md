- `GET /api/consumer/notifications/unread/count` - Fetch unread notification count
- `POST /api/consumer/notifications/mark-read` - Mark notifications as read

## Accessibility

- All stat cards are keyboard navigable with Enter/Space to activate
- Activity feed items have proper ARIA labels
- Notification count announced to screen readers
- Skip link to main content available
- Left navigation has proper ARIA landmarks
- Color contrast meets WCAG AA standards
