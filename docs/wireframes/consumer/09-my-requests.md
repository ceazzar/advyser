## Components

### Tab Navigation
- Type: tab bar
- Data fields: Tab label, count badge
- Actions: Click tab â†’ Filter requests by status
- States: active, inactive, disabled (if count is 0)

### Filter Dropdown
- Type: dropdown select
- Data fields: Label, options list
- Options: All Types, Financial Advisor, Property Advisor, Mortgage Broker, Tax Accountant
- Actions: Select option â†’ Filter requests by advisor type
- States: default, open, selected

### Request Card
- Type: card with horizontal layout
- Data fields:
  - Advisor avatar
  - Advisor name
  - Business name
  - Advisor type
  - Request date
  - Status badge
  - Last activity description
- Actions:
  - Click card â†’ Navigate to request detail
  - Click View Details â†’ Navigate to request detail
  - Click Message â†’ Navigate to conversation
- States: default, hover, loading

### Status Badge
- Type: badge/chip
- Variants:
  - New (blue) - Request submitted, awaiting response
  - Contacted (green) - Advisor has responded
  - Booked (purple) - Appointment scheduled
  - Completed (gray) - Consultation finished
- States: static (no interaction)

### Status Legend
- Type: info panel
- Data fields: Status color, label, description
- Actions: None (informational only)
- States: expanded (desktop), collapsible (mobile)

- Click "Active" tab â†’ Shows only New, Contacted, and Booked requests

- Click "All" tab â†’ Shows all requests regardless of status
- Select advisor type â†’ Filters list to show only that advisor type
- Click request card â†’ Navigates to /consumer/requests/{id}
- Click "View Details" â†’ Navigates to /consumer/requests/{id}
- Click "Message" â†’ Navigates to /consumer/messages/{conversationId}
- Click "Find an Advisor" (empty state) â†’ Navigates to /consumer/search
- Click "Clear Filters" (filtered empty state) â†’ Resets all filters
