# Client Workspace - Copilot Tab Wireframe

## Page Purpose
The Copilot tab is the AI-powered productivity hub for the client relationship. Advisors can input meeting transcripts or upload documents, and Copilot generates meeting summaries, action items, follow-up email drafts, client brief updates, and compliance flags. This is the core AI feature of the Advyser platform, designed to dramatically reduce post-meeting administrative work.

### CopilotRun
```typescript
interface CopilotRun {
  id: string;
  clientId: string;
  advisorId: string;
  input: {
    type: 'text' | 'file';
    content?: string; // For text input
    fileUrl?: string; // For file input
    fileName?: string;
  };
  context: {
    meetingType: 'discovery' | 'review' | 'strategy' | 'pre_approval' | 'other';
    meetingDate: Date;
    tone: 'formal' | 'friendly';
  };
  requestedOutputs: ('summary' | 'actions' | 'follow_up' | 'brief' | 'flags')[];

    summary?: CopilotOutput;
    actions?: CopilotOutput;
    followUp?: CopilotOutput;
    brief?: BriefUpdates;

    tasksCreated?: string[];
    briefUpdated?: boolean;
  };

  completedAt?: Date;
  error?: string;
}
```
