# Client Workspace - Notes Tab Wireframe

## Page Purpose
The Notes tab provides a structured repository for all advisor notes related to a client. Notes can be manually created or auto-generated by Copilot from meeting transcripts. The tab supports categorization, search, and version history to maintain a complete record of client interactions and advice.

## URL Pattern
`/advisor/clients/:id/notes`

## User Role

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/advisor/clients/:id/notes` | Fetch notes with filters/pagination |
| POST | `/api/advisor/clients/:id/notes` | Create new note |
| GET | `/api/advisor/clients/:id/notes/:noteId` | Fetch single note with full content |
| PUT | `/api/advisor/clients/:id/notes/:noteId` | Update note |
| DELETE | `/api/advisor/clients/:id/notes/:noteId` | Delete note |
| GET | `/api/advisor/clients/:id/notes/:noteId/versions` | Fetch version history |
| GET | `/api/advisor/clients/:id/notes/:noteId/original` | Fetch original transcript |

## Data Models

### Note
```typescript
interface Note {
  id: string;
  clientId: string;
  type: 'meeting_summary' | 'action_items' | 'file_note' | 'general';
  title: string;
  content: string; // Rich text / markdown
  preview: string; // First 200 chars
  source: 'manual' | 'copilot';
  copilotRunId?: string; // If generated by Copilot
  originalTranscript?: string; // If generated by Copilot
  author: {
    id: string;
    name: string;
  };
  versions: NoteVersion[];
  createdAt: Date;
  updatedAt: Date;
}
```

### NoteVersion
```typescript
interface NoteVersion {
  version: number;
  content: string;
  editedBy: {
    id: string;
    name: string;
    type: 'advisor' | 'copilot';
  };
  editedAt: Date;
}
```

## Accessibility Notes

- Note cards are focusable and navigable via keyboard
- Type and source badges have aria-labels
- Rich text editor supports keyboard shortcuts
- Modal dialog traps focus
- Search input has aria-live region for result count updates
- Version history list is navigable via arrow keys
