# Client Workspace - Documents Tab Wireframe

## Page Purpose
The Documents tab provides a centralized file management area for all documents related to a client relationship. Advisors can upload, organize, and share documents with visibility controls to distinguish between client-shared files and internal-only documents.

## URL Pattern
`/advisor/clients/:id/documents`

## User Role
advisor (authenticated)

## Components

### Upload Area
| Element | Description |
|---------|-------------|
| Drop zone | Dashed border area for drag & drop |

## Security Considerations

- All uploads scanned for malware before storage
- Files stored in secure cloud storage with signed URLs
- Download URLs expire after 1 hour
- Internal documents never exposed to client API
- File type validation on both client and server
- Maximum file size enforced server-side

## Accessibility Notes

- Upload area accepts keyboard focus and Enter/Space to open file picker
- Progress bars have aria-valuenow for screen readers
- File type icons have alt text
- Folder expansion state communicated via aria-expanded
- Delete confirmation is a modal dialog with focus trap
- Action buttons have descriptive aria-labels ("Download Tax_Return_2024.pdf")
