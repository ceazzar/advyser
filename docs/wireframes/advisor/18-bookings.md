## API Endpoints

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `GET /api/advisor/bookings` | GET | List all bookings |
| `GET /api/advisor/bookings/:id` | GET | Get booking details |
| `POST /api/advisor/bookings` | POST | Create new booking |
| `PATCH /api/advisor/bookings/:id` | PATCH | Update booking |
| `DELETE /api/advisor/bookings/:id` | DELETE | Cancel booking |
| `POST /api/advisor/bookings/:id/reschedule` | POST | Reschedule booking |
| `GET /api/advisor/availability` | GET | Get availability settings |
| `PUT /api/advisor/availability` | PUT | Update availability |
| `GET /api/advisor/calendar/sync` | GET | Get calendar sync status |
| `POST /api/advisor/calendar/connect` | POST | Connect external calendar |
| `DELETE /api/advisor/calendar/disconnect` | DELETE | Disconnect calendar |

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `from` | date | Start date for range |
| `to` | date | End date for range |
| `status` | string | Filter by status (upcoming, past, cancelled) |
| `type` | string | Filter by booking type |
| `consumerId` | string | Filter by consumer |

## Calendar Integration

### Google Calendar
- OAuth 2.0 authentication
- Two-way sync (read existing events, write new bookings)
- Automatic conflict detection
- Meeting link generation via Google Meet

### Outlook Calendar
- OAuth 2.0 authentication
- Two-way sync
- Microsoft Teams integration for video calls

## Accessibility

- Calendar is keyboard navigable (arrow keys for day navigation)
- Booking blocks have ARIA labels with full booking details
- Time slots announce availability status
- Color coding supplemented with text labels
- Modal focus management
- Screen reader announces upcoming meeting alerts
