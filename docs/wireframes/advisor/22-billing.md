## Stripe Integration

### Payment Flow
1. User clicks "Update Payment Method"
2. Modal opens with Stripe Elements form
3. Card details submitted to Stripe
4. Stripe returns payment method token
5. Token sent to backend
6. Backend attaches payment method to customer
7. Modal closes, payment method displayed

### Subscription Flow
1. User selects plan from comparison modal
2. Confirmation modal shows pricing details
3. User confirms upgrade
4. Backend creates/updates Stripe subscription
5. Stripe charges payment method (prorated)
6. Webhook updates subscription status
7. UI updates to show new plan
