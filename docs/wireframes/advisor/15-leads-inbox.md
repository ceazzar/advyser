## API Endpoints

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `GET /api/advisor/leads` | GET | Fetch all leads with filters |
| `GET /api/advisor/leads/:id` | GET | Fetch single lead detail |
| `PATCH /api/advisor/leads/:id/status` | PATCH | Update lead status |
| `POST /api/advisor/leads/:id/decline` | POST | Decline with reason |
| `POST /api/advisor/leads/bulk/status` | POST | Bulk status update |
| `POST /api/advisor/leads/bulk/archive` | POST | Bulk archive |
| `GET /api/advisor/leads/stats` | GET | Get lead counts by status |

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `status` | string | Filter by status (comma-separated) |
| `dateFrom` | date | Filter from date |
| `dateTo` | date | Filter to date |
| `source` | string | Filter by source |
| `search` | string | Search by name/goal |
| `sort` | string | Sort field (date, name, status) |
| `order` | string | Sort order (asc, desc) |
| `page` | number | Page number |
| `limit` | number | Items per page |
