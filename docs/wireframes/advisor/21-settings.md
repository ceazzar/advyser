## API Endpoints

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `GET /api/advisor/settings` | GET | Fetch all settings |
| `PUT /api/advisor/settings/account` | PUT | Update account settings |
| `PUT /api/advisor/settings/notifications` | PUT | Update notification prefs |
| `PUT /api/advisor/settings/service-areas` | PUT | Update service areas |
| `PUT /api/advisor/settings/lead-preferences` | PUT | Update lead preferences |
| `PUT /api/advisor/settings/privacy` | PUT | Update privacy settings |
| `POST /api/advisor/settings/email/change` | POST | Initiate email change |
| `POST /api/advisor/settings/password/change` | POST | Change password |
| `POST /api/advisor/settings/2fa/enable` | POST | Enable 2FA |
| `DELETE /api/advisor/settings/2fa` | DELETE | Disable 2FA |
| `GET /api/advisor/integrations` | GET | List integrations |
| `POST /api/advisor/integrations/:provider/connect` | POST | Connect integration |
| `DELETE /api/advisor/integrations/:provider` | DELETE | Disconnect integration |
| `POST /api/advisor/data/export` | POST | Request data export |
| `POST /api/advisor/account/deactivate` | POST | Deactivate account |
| `DELETE /api/advisor/account` | DELETE | Delete account |

## Data Models

### AdvisorSettings
```typescript
interface AdvisorSettings {
  account: {
    email: string;
    phone: string;
    twoFactorEnabled: boolean;
    twoFactorMethod?: 'app' | 'sms';
  };
  notifications: {
    email: {
      newLead: boolean;
      newMessage: boolean;
      bookingConfirmation: boolean;
      bookingReminder: boolean;
      bookingReminderTime: number; // minutes before
      weeklySummary: boolean;
      weeklySummaryDay: string;
    };
    push: {
      newLead: boolean;
      newMessage: boolean;
      bookingReminder: boolean;
    };
    sms: {
      enabled: boolean; // Pro only
      newLead: boolean;
      bookingReminder: boolean;
    };
  };
  serviceAreas: {
    videoEnabled: boolean;
    phoneEnabled: boolean;
    inPersonEnabled: boolean;
    officeAddress?: string;
    serviceRadius: number; // km
    postcodes: string[];
  };
  leadPreferences: {
    autoAccept: boolean;
    minimumBudget?: number;
    preferredModes: ('video' | 'phone' | 'in-person')[];
    maxLeadsPerWeek?: number;
  };
  privacy: {
    showInSearch: boolean;
    showAddress: boolean;
    showPhone: boolean;
    allowAnalytics: boolean;
    receivePlatformUpdates: boolean;
  };
}

interface Integration {
  provider: 'google_calendar' | 'outlook_calendar' | 'zoom' | 'teams';
  connected: boolean;
  email?: string;
  connectedAt?: Date;
  lastSyncedAt?: Date;
  settings: Record<string, boolean>;
}
```

## Accessibility

- Settings navigation is keyboard accessible
- Toggle switches have proper ARIA labels
- Form validation errors linked to inputs
- Modal focus trap and Escape to close
- Danger zone actions require confirmation
- Password strength indicator has text alternatives
- Color not sole indicator of status (icons included)
