## Components

### Profile Header
- Type: `section/header`
- Data fields:
  - `avatar_url` (string)
  - `name` (string)
  - `business_name` (string)
  - `advisor_type` (enum)
  - `verified` (boolean)
  - `verified_date` (date)
  - `location` (suburb, state, postcode)
  - `rating` (number)
  - `review_count` (number)
- Actions: None (display only)
- States:
  - Default
  - Loading (skeleton)

### Verified Badge with Tooltip
- Type: `badge/tooltip`
- Data fields:
  - `verified` (boolean)
  - `verified_date` (date)
  - `verification_source` (string)
- Actions:
  - Hover/tap - shows tooltip with details
- States:
  - Verified (green checkmark)
  - Not verified (no badge shown)

### Credentials Section
- Type: `card/information`
- Data fields:
  - `afsl_number` (string)
  - `ar_number` (string, optional)
  - `licensee_name` (string)
  - `registration_status` (enum: active, inactive, suspended)
  - `verified_date` (date)
- Actions:
  - [View on ASIC] - external link to ASIC register
- States:
  - Active (green indicator)
  - Inactive/Suspended (red indicator, warning)

### About Section
- Type: `section/content`
- Data fields:
  - `bio` (text, rich text)
  - `who_i_help` (array of strings)
  - `who_i_dont_work_with` (array of strings)
- Actions:
  - [Read More] - expands truncated content (mobile)
- States:
  - Default
  - Expanded (mobile)

### Specialties Tags
- Type: `tag/display`
- Data fields:
  - `specialties` (array of strings)
- Actions:
  - Click tag - navigates to search filtered by specialty
- States:
  - Default
  - Hover (clickable indication)

### Services & Pricing Table
- Type: `table/pricing`
- Data fields:
  - Array of services:
    - `name` (string)
    - `description` (string)
    - `duration` (string)
    - `price` (string, formatted)
- Actions:
  - [View All Services] - expands on mobile
- States:
  - Default
  - Expanded (mobile)

### Rating Summary
- Type: `section/statistics`
- Data fields:
  - `average_rating` (number)
  - `total_reviews` (number)
  - `rating_distribution` (object: 5 star count, 4 star count, etc.)
- Actions: None (display only)
- States:
  - Has reviews
  - No reviews yet

### Review Card
- Type: `card/review`
- Data fields:
  - `rating` (number 1-5)
  - `text` (string)
  - `reviewer_name` (string, partial)
  - `service_type` (string)
  - `date` (date)
  - `verified_engagement` (boolean)
- Actions: None (display only)
- States:
  - Default
  - Expanded (if long review)

### Action Sidebar
- Type: `sidebar/sticky`
- Data fields:
  - `accepting_clients` (boolean)
  - `next_available` (datetime)
  - `response_time_hours` (number)
  - `languages` (array of strings)
  - `service_modes` (array: in_person, video, phone)
- Actions:
  - [Request Intro] - navigates to `/request-intro/{slug}`
  - [Book Consultation] - opens booking modal/external link
  - [Send Message] - opens message modal (requires auth)
  - [Copy Link] - copies profile URL
  - [Email] - opens mailto with profile link
- States:
  - Available
  - Not accepting clients (grayed CTAs)
  - Loading

### Similar Advisors
- Type: `section/carousel`
- Data fields:
  - Array of advisor preview cards
- Actions:
  - [View Profile] - navigates to advisor
- States:
  - Has similar advisors
  - No similar advisors (hidden)

## AU Compliance Notes

1. **Credential Verification Display**
   - AFSL/AR number must be prominently displayed
   - Link to ASIC register for independent verification
   - Clear indication of verification date and source

2. **Registration Status**
   - Must show if advisor is "Active" on ASIC register
   - If suspended/inactive, must show warning and disable contact CTAs

3. **Price Disclaimer**
   - "Prices are indicative and may vary based on complexity"
   - "Final fees will be confirmed in writing before any advice is provided"

4. **Review Authenticity**
   - "All reviews are from verified engagements through Advyser"
   - "We do not import reviews from other platforms"
   - Reviews tied to actual service engagements

5. **No Advice Endorsement**
   - Platform does not endorse quality of advice
   - Rating is based on client feedback, not advice quality assessment

6. **Professional Designations**
   - CFP, FPA, AFA memberships can be displayed if verified
   - Must link to relevant professional body for verification
