## Components

### Progress Indicator
- Type: `navigation/stepper`
- Data fields:
  - `current_step` (number 1-3)
  - `steps` (array of step labels)
- Actions: None (display only, no backward navigation via stepper)
- States:
  - Step incomplete (â—‹)
  - Step complete (âœ“)
  - Step active (â—)

### Goal Textarea
- Type: `form/textarea`
- Data fields:
  - `goal_text` (string, 500 char max)
  - `character_count` (number)
- Actions:
  - Type - updates character count
- States:
  - Default
  - Focused
  - Error (empty on continue)
  - Character limit warning (>450 chars)

### Goal Tags (Checkboxes)
- Type: `form/checkbox-group`
- Data fields:
  - `selected_goals` (array of strings)
  - `available_goals` (array from advisor specialties)
- Actions:
  - Toggle checkbox - adds/removes from selection
- States:
  - Unchecked
  - Checked
  - Collapsed (mobile, show more)

### Timeline Dropdown
- Type: `form/select`
- Data fields:
  - `selected_timeline` (string)
  - `options` (predefined array)
- Actions:
  - Select option - updates value
- States:
  - Default (placeholder)
  - Selected
  - Error (required, empty)

### Budget Range Dropdown
- Type: `form/select`
- Data fields:
  - `selected_range` (string, optional)
  - `options` (predefined AU ranges)
- Actions:
  - Select option - updates value
- States:
  - Default (placeholder)
  - Selected
  - Skipped (prefer not to say)

### Meeting Mode Radio
- Type: `form/radio-group`
- Data fields:
  - `selected_mode` (string)
  - `available_modes` (from advisor profile)
- Actions:
  - Select option - updates value
- States:
  - Unselected
  - Selected

### Location Input
- Type: `form/autocomplete`
- Data fields:
  - `location` (string)
  - `suburb`, `state`, `postcode` (parsed)
- Actions:
  - Type - triggers autocomplete
  - Select suggestion - populates field
- States:
  - Default
  - Suggestions showing
  - Selected
  - Error (invalid location)

### File Upload
- Type: `form/file-upload`
- Data fields:
  - `files` (array of File objects)
  - `max_files` (5)
  - `max_size_mb` (10)
  - `allowed_types` (PDF, DOC, DOCX, JPG, PNG)
- Actions:
  - Drag file - initiates upload
  - Click browse - opens file picker
  - Click [Ã—] - removes file
- States:
  - Empty
  - Uploading (progress bar)
  - Files uploaded (list)
  - Error (too large, wrong type)

### Review Summary Cards
- Type: `card/summary`
- Data fields:
  - Step data (compiled from previous steps)
- Actions:
  - [Edit] - returns to that step
- States:
  - Default

### Contact Form
- Type: `form/contact`
- Data fields:
  - `name` (string, required)
  - `email` (string, required, validated)
  - `phone` (string, optional, AU format)
  - `create_account` (boolean)
- Actions:
  - Input - standard form input
- States:
  - Default
  - Pre-filled (if logged in)
  - Validation error

### Consent Checkboxes
- Type: `form/checkbox-group`
- Data fields:
  - `consent_share` (boolean, required)
  - `consent_terms` (boolean, required)
- Actions:
  - Toggle - updates consent state
- States:
  - Unchecked
  - Checked
  - Error (required, unchecked)

### Advisor Sidebar
- Type: `sidebar/static`
- Data fields:
  - Advisor preview (avatar, name, type, verified, location, rating)
  - Specialties (array)
  - Response time
- Actions: None (display only)
- States:
  - Default
  - Collapsed (mobile - shown as header card)

## AU Compliance Notes

1. **Explicit Consent**
   - User must explicitly consent to share information with named advisor
   - Consent checkbox cannot be pre-checked
   - Clear statement of what will be shared

2. **Privacy Policy Link**
   - Must link to full privacy policy
   - Must explain how data is used and stored

3. **Data Minimisation**
   - Only collect data necessary for the inquiry
   - Asset range is optional
   - Phone is optional

4. **No Financial Advice**
   - Form does not constitute advice
   - Platform is facilitating connection only
   - Advisor will provide any advice directly

5. **Document Security**
   - Uploaded documents are encrypted at rest
   - Only shared with consented advisor
   - Retention policy disclosed in privacy policy
