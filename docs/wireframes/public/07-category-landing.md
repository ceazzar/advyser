# Category Landing Page Wireframe (SEO)

## Page Purpose
SEO-optimized landing pages for specific advisor categories in specific locations. These pages serve as entry points from search engines and help users find relevant advisors while providing valuable informational content for ranking purposes.

## URL Pattern
`/australia/{state}/{city}/{category}`

Examples:
- `/australia/nsw/sydney/financial-advisers`
- `/australia/vic/melbourne/mortgage-brokers`
- `/australia/qld/brisbane/buyers-agents`
- `/australia/wa/perth/property-advisors`

Also supports suburb-level:
- `/australia/nsw/sydney/bondi/financial-advisers`

## User Role
public (optimized for search engine discovery)

## Components

### Breadcrumbs
- Type: `navigation/breadcrumb`
- Data fields:
  - `crumbs` (array of {label, url})
- Actions:
  - Click crumb - navigates to that level
- States:
  - Default
  - Current page (not clickable)

### Page Header (H1 + Intro)
- Type: `section/header`
- Data fields:
  - `h1` (string, includes location + category)
  - `intro_text` (string, SEO-optimized)
  - `advisor_count` (number)
- Actions: None
- States: Static

### Search Bar (Pre-filled)
- Type: `form/search`
- Data fields:
  - `location` (pre-filled from URL)
  - `advisor_type` (pre-filled from URL)
- Actions:
  - [Search] - navigates to /search with params
- States:
  - Pre-filled (from URL path)
  - Editable

### Featured Advisor Card (Extended)
- Type: `card/advisor-featured`
- Data fields:
  - All standard advisor fields
  - Extended bio snippet
  - Full specialty list
- Actions:
  - [View Profile] - navigates to advisor profile
  - [Request Introduction] - navigates to request intro
- States:
  - Default
  - Hover

### Advisor List Row (Compact)
- Type: `list/advisor-row`
- Data fields:
  - Avatar, name, business, verified badge
  - Location (suburb)
  - Rating + review count
  - Top 3 specialties
  - Price tier
- Actions:
  - Click row - navigates to profile
  - [View Profile] - navigates to profile
- States:
  - Default
  - Hover

### Pagination
- Type: `navigation/pagination`
- Data fields:
  - `current_page`, `total_pages`, `total_results`
- Actions:
  - Click page - loads that page
- States:
  - Standard pagination states

### SEO Content Section
- Type: `section/content`
- Data fields:
  - `title` (string)
  - `content` (rich text, HTML)
  - `specialty_counts` (array)
- Actions: None
- States: Static (collapsed on mobile with "Read More")

### FAQ Accordion
- Type: `accordion/faq`
- Data fields:
  - `faqs` (array of {question, answer})
- Actions:
  - Click question - expands/collapses answer
- States:
  - Collapsed (default)
  - Expanded
- Schema markup: `FAQPage` JSON-LD

### Related Searches
- Type: `section/links`
- Data fields:
  - `nearby_suburbs` (array of {label, url})
  - `other_types` (array of {label, url})
  - `other_cities` (array of {label, url})
- Actions:
  - Click link - navigates to that landing page
- States:
  - Default
  - Hover

## Interactions

| Trigger | Action |
|---------|--------|
| Click breadcrumb | Navigate to that level (e.g., NSW â†’ all NSW advisors) |

| Click [Request Introduction] | Navigate to `/request-intro/{slug}` |

## AU Compliance Notes

1. **No Advice Claims**
   - Page provides information about advisors, not financial advice
   - Disclaimer in footer applies

2. **Accurate Advisor Counts**
   - Counts must reflect actual verified advisors
   - Update counts regularly (cache with TTL)

3. **Verified Status Clarity**
   - "Verified" means credentials checked against ASIC register
   - Not an endorsement of advice quality

4. **ASIC Licence Links**
   - Featured advisors should link to their ASIC register entry
   - Builds trust and supports due diligence

5. **No Misleading Rankings**
   - "Featured" is based on engagement/completeness, not payment
   - If paid placement, must be disclosed as "Sponsored"

## Performance Considerations

1. **Static Generation**
   - Pre-generate top 100 landing pages at build time
   - Use ISR (Incremental Static Regeneration) for others

2. **Caching**
   - Cache advisor counts for 1 hour
   - Cache featured advisors for 15 minutes
   - Full page cache with CDN

3. **Image Optimization**
   - Lazy load advisor avatars below fold
   - Use WebP with fallback
   - Responsive images for mobile

4. **Core Web Vitals**
   - Target LCP < 2.5s
   - Target CLS < 0.1
   - Minimize layout shifts from dynamic content
