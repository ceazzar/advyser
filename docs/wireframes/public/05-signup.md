## Page Purpose

## URL Pattern
`/signup`
`/signup?role=advisor` (pre-selects advisor tab)

## User Role

## Components

### Tab Selector
- Type: `navigation/tabs`
- Data fields:
  - `active_tab` (consumer | advisor)
- Actions:
  - Click tab - switches form content
- States:
  - Consumer active (underlined)
  - Advisor active (underlined)

### Email Input
- Type: `form/input`
- Data fields:
  - `email` (string)
- Actions:
  - Input - validates email format
  - Blur - check if email already exists
- States:
  - Default
  - Focused
  - Valid (green checkmark)
  - Error (invalid format or already exists)

### Password Input
- Type: `form/password`
- Data fields:
  - `password` (string)
  - `show_password` (boolean)
- Actions:
  - Input - validates strength
  - Click [ğŸ‘] - toggle visibility
- States:
  - Default
  - Focused
  - Weak (red indicator)
  - Medium (yellow indicator)
  - Strong (green indicator)
  - Error (doesn't meet requirements)

### Name Input
- Type: `form/input`
- Data fields:
  - `full_name` (string)
- Actions:
  - Input - standard
- States:
  - Default
  - Focused
  - Error (too short)

### Phone Input
- Type: `form/phone`
- Data fields:
  - `country_code` (default +61)
  - `phone_number` (string)
- Actions:
  - Input - formats as AU phone number
- States:
  - Default
  - Focused
  - Valid
  - Error (invalid format)

### Business Name Input (Advisor only)
- Type: `form/input`
- Data fields:
  - `business_name` (string)
- Actions:
  - Input - standard
- States:
  - Default
  - Focused
  - Error (required)

### Advisor Type Dropdown (Advisor only)
- Type: `form/select`
- Data fields:
  - `advisor_type` (enum)
  - `options` (Financial Adviser, Mortgage Broker, Buyer's Agent, Property Advisor)
- Actions:
  - Select - updates value
- States:
  - Default (placeholder)
  - Selected
  - Error (required)

### Terms Checkbox
- Type: `form/checkbox`
- Data fields:
  - `terms_accepted` (boolean)
- Actions:
  - Click - toggles
  - Click [Terms of Service] - opens in new tab
  - Click [Privacy Policy] - opens in new tab
- States:
  - Unchecked
  - Checked
  - Error (required but unchecked)

### Marketing Checkbox (Consumer only)
- Type: `form/checkbox`
- Data fields:
  - `marketing_consent` (boolean)
- Actions:
  - Click - toggles
- States:
  - Unchecked (default)
  - Checked

### Licence Confirmation Checkbox (Advisor only)
- Type: `form/checkbox`
- Data fields:
  - `licence_confirmed` (boolean)
- Actions:
  - Click - toggles
- States:
  - Unchecked
  - Checked
  - Error (required but unchecked)

### Submit Button
- Type: `button/submit`
- Data fields: None
- Actions:
  - Click - validates form, submits if valid
- States:
  - Default
  - Hover
  - Loading (spinner)
  - Disabled (form invalid)

### OAuth Button (Google)
- Type: `button/oauth`
- Data fields: None
- Actions:
  - Click - initiates Google OAuth flow
- States:
  - Default

## Post-Signup Flows

### Consumer
1. Account created
2. Email verification sent (optional, can browse without)
3. Redirect to `/search` or previous page (if came from lead form)

### Advisor
1. Account created
2. Email verification sent (required for full access)
3. Redirect to `/advisor/onboarding` to:
   - Claim existing business listing OR
   - Create new business profile
   - Enter AFSL/licence details
   - Begin verification process

## AU Compliance Notes

1. **Consent for Data Collection**
   - Clear consent checkbox for terms and privacy
   - Cannot be pre-checked
   - Must agree before account creation

2. **Marketing Consent (Spam Act 2003)**
   - Marketing checkbox separate from terms
   - Not pre-checked
   - Clear opt-out language
   - Must be able to unsubscribe

3. **Adviser Licence Confirmation**
   - Advisors must confirm they hold appropriate licence
   - Platform will verify this during onboarding
   - Account may be restricted if verification fails

4. **Privacy Collection Notice**
   - Privacy policy must be accessible before signup
   - Must explain how data is used, stored, shared

5. **Password Requirements**
   - Minimum 8 characters
   - Mix of letters and numbers
   - Industry standard security
