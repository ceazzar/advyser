## Components

### Navigation Bar
- Type: `header/navigation`
- Data fields:
  - Logo (link to home)
  - Navigation links array
  - Auth state (logged in/out)
- Actions:
  - [Login] button - navigates to /login
  - [Sign Up] button - navigates to /signup
- States:
  - Default (logged out)
  - Authenticated (shows user avatar + dropdown)
  - Mobile (hamburger menu)

### Hero Search
- Type: `form/search`
- Data fields:
  - Location input (autocomplete with Google Places API)
  - Advisor type dropdown
- Actions:
  - [Search] button - submits to /search with query params
  - [Use my location] - requests geolocation permission
- States:
  - Default
  - Location loading (spinner in input)
  - Location error (fallback to manual entry)

### Category Tile
- Type: `card/link`
- Data fields:
  - Icon
  - Category name
  - Description
  - Advisor count (dynamic)
- Actions:
  - Click - navigates to /search?type=[category]
- States:
  - Default
  - Hover (slight elevation)

### Trust Strip
- Type: `section/static`
- Data fields:
  - Three trust indicators with icons
- Actions: None
- States: Static (no interaction)

### How It Works
- Type: `section/informational`
- Data fields:
  - Step number
  - Icon
  - Title
  - Description
- Actions:
  - [Get Started] - navigates to search
- States: Static

### Featured Advisor Card
- Type: `card/advisor-preview`
- Data fields:
  - Avatar URL
  - Name
  - Advisor type
  - Location (suburb, state)
  - Verified badge (boolean)
  - Rating (number)
  - Review count
- Actions:
  - [View Profile] - navigates to /advisor/[slug]
- States:
  - Default
  - Hover
  - Loading skeleton

### For Advisors CTA
- Type: `section/cta`
- Data fields:
  - Headline
  - Description
  - CTA button text
- Actions:
  - [Claim Your Profile] - navigates to /signup?role=advisor
- States: Static

### Footer
- Type: `footer/global`
- Data fields:
  - Logo
  - Link groups (4 columns)
  - Disclaimer text
  - ABN
  - Social links
- Actions:
  - All links navigate to respective pages
- States: Static

## Interactions

| Trigger | Action |
|---------|--------|
| Click [Search] | Navigate to `/search?location={loc}&type={type}` |
| Click Category Tile | Navigate to `/search?type={category}` |
| Click [View Profile] on advisor card | Navigate to `/advisor/{slug}` |
| Click [Get Started] | Navigate to `/search` |
| Click [Claim Your Profile] | Navigate to `/signup?role=advisor` |
| Click [Login] | Navigate to `/login` |
| Click [Sign Up] | Navigate to `/signup` |
| Press Enter in search bar | Submit search |
| Click [Use my location] | Request browser geolocation |

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/categories` | Fetch category list with advisor counts |
| GET | `/api/advisors/featured` | Fetch featured advisors for homepage |
| GET | `/api/locations/autocomplete?q={query}` | Location autocomplete suggestions |

## AU Compliance Notes

1. **Footer Disclaimer (Required)**
   - Must clearly state Advyser does not provide advice
   - Must indicate advisors operate under their own AFSL
   - Must be prominently displayed

2. **ABN Display**
   - Australian Business Number must be visible in footer

3. **No Advice Claims**
   - Hero and marketing copy must not imply Advyser provides advice
   - "Connect with" language preferred over "get advice from"

4. **Verified Badge Context**
   - Trust strip should clarify what "verified" means (licence checked, not quality endorsed)
